# VLAN Design

This file will document the VLAN and subnet structure for the homelab.

## Current VLANs

- VLAN 10 – Server / Compute (10.10.10.0/24)
- VLAN 20 – Home Wired (10.10.20.0/24)
- VLAN 30 – Home Wi-Fi (10.10.30.0/24)
- VLAN 40 – Guest Wi-Fi (10.10.40.0/24)
- VLAN 50 – Lab / Pi / Automation (10.10.50.0/24)
- VLAN 60 – Management (10.10.60.0/24)

More details to be added later.
