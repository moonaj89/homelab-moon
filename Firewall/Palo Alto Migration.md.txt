# PA-440 + Cisco 3560 VLAN Migration Runbook

> Status: In progress  
> Last updated: YYYY-MM-DD

This runbook documents the migration from a directly-connected PA-440 design to a switch-based, VLAN-aware architecture using a Cisco 3560.

## Goals

- Introduce enterprise-style VLAN segmentation
- Move hosts behind a core switch
- Prepare for a NUC-based virtualization host and NAS
- Keep rollback simple and low-risk

## Phases

- Phase 0 – Preparation
- Phase 1 – Switch configuration
- Phase 2 – PA-440 subinterfaces and DHCP
- Phase 3 – Security policy
- Phase 4 – Device migration & validation
